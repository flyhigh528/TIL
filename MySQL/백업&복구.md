# Backup
***

## Backup의 종류
- Hot Backup
  > 데이터베이스 구동 중 Backup
  > ex) $> mysqldump -u root -p 'database명' > output.sql
- Cold Backup
  > 데이터베이스 Stop 후 Backup
- Physical Backup
  > Datafile Backup
- Logical Backup


***
## Daily Backup Shell Script 작성해보기
#!/bin/bash

DATE='date +%T%m%d --date=yesterday'
FILENAME=${DATE}.sql

cd /백업폴더경로이동
mysqldump -uroot -p패스워드 디비명 > $FILENAME
